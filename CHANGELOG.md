# Changelog

Все значимые изменения в проекте SprutNet будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2024-12-19

### 🎉 Первый релиз MVP

#### ✨ Добавлено
- **Полная интеграция с Maersk API** для получения данных о рейсах, судах и портах
- **Современный UI/UX** на базе shadcn/ui компонентов
- **Система поиска портов** с автодополнением и фильтрацией
- **Планировщик маршрутов** с анализом времени, стоимости и оптимальности
- **База данных** с просмотром портов, судов и расписаний
- **Поддержка тем** (светлая/темная) с ThemeToggle
- **Адаптивный дизайн** для всех устройств
- **TypeScript** типизация всего проекта
- **Prisma ORM** для работы с базой данных
- **Next.js 15** с App Router
- **Tailwind CSS 3.4** для стилизации
- **React Hook Form** с Zod валидацией

#### 🎨 UI/UX Компоненты
- Button, Card, Badge, Separator
- Tabs, Table, Skeleton, Progress
- Alert, Dialog, Dropdown Menu
- Form компоненты с валидацией
- Theme Toggle для переключения тем
- Иконки Lucide React

#### 🌐 API Endpoints
- `GET /api/schedules` - поиск рейсов
- `GET /api/ports/search` - поиск портов
- `GET /api/vessels` - список судов
- `GET /api/locations` - список локаций
- `GET /api/ocean-products` - расписания
- `GET /api/deadlines` - дедлайны
- `POST /api/load-data` - загрузка данных
- `POST /api/seed-data` - заполнение тестовыми данными

#### 📱 Страницы
- **Главная страница** (`/`) - обзор возможностей и статистика
- **Планировщик** (`/planner`) - поиск и анализ маршрутов
- **База данных** (`/data`) - просмотр и управление данными

#### 🛠 Технические особенности
- **Monorepo структура** с pnpm workspaces
- **Shared пакет** для общих типов и утилит
- **Prisma схема** для базы данных
- **ESLint и Prettier** для качества кода
- **TypeScript** для типизации
- **Tailwind CSS** для стилизации
- **shadcn/ui** для компонентов

#### 🔒 Безопасность
- Валидация всех входных данных через Zod
- Защищенные API endpoints
- Переменные окружения для конфиденциальных данных
- TypeScript для предотвращения ошибок типов

#### 📚 Документация
- Подробный README.md с инструкциями по установке
- CHANGELOG.md для отслеживания изменений
- MIT лицензия
- Структура проекта и API документация

#### 🚀 Развертывание
- Готовность к деплою на Vercel
- Docker поддержка
- CI/CD готовность
- Переменные окружения настроены
