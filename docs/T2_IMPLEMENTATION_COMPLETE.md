# T2. Search Form & Basic Results - Реализация завершена ✅

## 🎯 Цель
Сделать форму поиска POL→POD + список рейсов.

## ✅ Выполненные задачи

### 1. Создана страница планировщика (`/planner`)
- **Современный дизайн** с использованием Tailwind CSS
- **Адаптивная верстка** для мобильных устройств
- **Интеграция с API** для поиска портов и расписаний
- **Обработка состояний** (загрузка, ошибки, пустые результаты)

### 2. Реализована форма поиска
- **Combobox для портов** (POL/POD) с автопоиском
- **DateRangePicker** для выбора дат отправления
- **Debounced поиск** портов (300ms задержка)
- **Валидация** обязательных полей
- **Интеграция с API** `/api/ports/search`

### 3. Создан компонент результатов
- **Карточки рейсов** с полной информацией
- **Сортировка** по трем критериям:
  - Самый ранний (по дате отправления)
  - Самый быстрый (по длительности маршрута)
  - Лучший выбор (баланс скорости и стоимости)
- **Подсветка лучших рейсов** с бейджами
- **Детальная информация** о судне, маршруте, расписании

### 4. Обновлена главная страница
- **Современный дизайн** с градиентным фоном
- **Hero секция** с призывом к действию
- **Карточки функций** с иконками
- **Ссылки на планировщик** и просмотр данных
