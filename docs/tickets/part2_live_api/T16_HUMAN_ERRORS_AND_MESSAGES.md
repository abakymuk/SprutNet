# T16. Человечные ошибки и сообщения

**Goal:** Понятные пользователю ошибки вместо кодов и стеков.

**Scope:**
- In: словарь сообщений для 400/404/429/5xx, кнопка-фоллбек на моки, повтор запроса.
- Out: доверие и предсказуемость поведения.

**AC (Gherkin):**
- Given 400/404, Then показываю "Не найдено расписаний по выбранным портам/датам…".
- Given 429, Then "Слишком много запросов. Повторим автоматически.". 
- Given 5xx, Then "Проблема на стороне поставщика…" + кнопка моки.

**DoD:**
- [ ] Сообщения локализованы/скрыты за словарём.
- [ ] Единый компонент ErrorState/Skeleton.
- [ ] e2e: сценарии ошибок покрыты.
