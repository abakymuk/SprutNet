# T14. Нормализация времени и таймзон

**Goal:** Единая модель времени: хранить UTC, показывать Local (POL/POD), избегать путаницы.

**Scope:**
- In: утилиты для конвертации UTC→Local с учётом таймзоны порта, двойной рендер (Local + мелким шрифтом UTC).
- Out: консистентный показ времени в карточках и дедлайнах.

**AC (Gherkin):**
- Given дедлайн с UTC, When рендерю Local по таймзоне POL, Then пользователь видит локальное время и UTC-хинт.
- Given API отдаёт `localTime` и `timeZone`, Then используем их как источник истины.

**DoD:**
- [ ] Утилиты времени покрыты unit-тестами (краевые случаи DST).
- [ ] UI показывает Local и UTC единообразно.
- [ ] Документация: правила отображения времени.
