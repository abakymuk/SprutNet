**Ключевые функции:**

#### Алгоритмы расчета
```typescript
// Средний транзит
const avgTransitTime = Math.round(
  transitTimes.reduce((sum, time) => sum + time, 0) / transitTimes.length
);

// Частота рейсов (за последние 4 недели)
const sailingsPerWeek = Math.round((recentSailings.length / 4) * 10) / 10;

// Надежность
const reliability = Math.round((onTimeSailings / sailings.length) * 100);

// Эффективность маршрута
const routeEfficiency = Math.min(100, Math.round(
  ((100 - avgTransitTime * 2) + (sailingsPerWeek * 10) + reliability) / 3
));
```
