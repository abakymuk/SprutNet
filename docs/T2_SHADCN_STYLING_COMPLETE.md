## 🎨 Дизайн-система

### Цветовая палитра
- **Primary**: Синие тона для основных действий
- **Secondary**: Нейтральные серые для второстепенных элементов
- **Muted**: Светлые серые для плейсхолдеров и вспомогательного текста
- **Border**: Консистентные границы для всех компонентов

### Типографика
- **Заголовки**: `text-foreground` с правильной иерархией
- **Основной текст**: `text-foreground` для читаемости
- **Вспомогательный текст**: `text-muted-foreground` для контекста
- **Ссылки**: `text-primary` для интерактивности

### Компоненты
- **Button**: Консистентные варианты (`default`, `outline`, `secondary`)
- **Card**: Единая структура с `bg-card` и `border-border`
- **Badge**: Семантические варианты для статусов
- **Input**: Правильные стили с `border-input` и `ring-ring`
