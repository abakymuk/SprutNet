### 3. Алгоритмы и логика

#### Система сравнения рейсов
```typescript
// Расчет позиций по различным критериям
- Цена (30% веса)
- Скорость (30% веса) 
- Надежность (20% веса)
- Популярность (20% веса)
```

#### Генерация рекомендаций
- ✅ **Самый ранний**: рейс с самой ранней датой отправления
- ✅ **Самый быстрый**: рейс с самым коротким транзитом
- ✅ **Самый дешевый**: рейс с лучшей ценой
- ✅ **Сбалансированный**: рейс с общим баллом > 80
- ✅ **Надежный**: рейс с высокой позицией по надежности
