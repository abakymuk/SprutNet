# Отчет об исправлениях API Endpoints

## 🎯 Цель
Исправить проблемы с API endpoints, которые возвращали неправильный формат данных или ошибки.

## ✅ Исправленные проблемы

### 1. **`/api/vessels/[imo]` - Отсутствовали поля `imo` и `name`**

**Проблема:**
- API возвращал только объект `vessel`, но тесты ожидали поля `imo` и `name` на верхнем уровне

**Решение:**
- Добавлены поля `imo` и `name` в ответ API
- Теперь API возвращает:
```json
{
  "imo": "1234567",
  "name": "MAERSK SEVILLE",
  "vessel": { /* полные данные судна */ },
  "source": "maersk|mock"
}
```

**Файлы изменены:**
- `apps/web/src/app/api/vessels/[imo]/route.ts`

### 2. **`/api/ports/search` - Отсутствовало поле `ports`**

**Проблема:**
- API возвращал только поле `data`, но тесты ожидали также поле `ports`

**Решение:**
- Добавлено поле `ports` в тип `PortSearchResponse`
- API теперь возвращает:
```json
{
  "success": true,
  "ports": [...], // Для обратной совместимости
  "data": [...],  // Основные данные
  "total": 10
}
```

**Файлы изменены:**
- `apps/web/src/lib/types/ports.ts`
- `apps/web/src/app/api/ports/search/route.ts`

### 3. **`/api/load-data` - Ошибка 500 при недоступности Maersk API**

**Проблема:**
- API возвращал ошибку 500 при недоступности Maersk API
- Не было fallback механизма

**Решение:**
- Добавлена проверка подключения к базе данных
- Улучшена обработка ошибок Maersk API
- Добавлен fallback на mock данные
- API теперь возвращает:
```json
{
  "success": true,
  "message": "Maersk API недоступен. Данные не загружены.",
  "count": 0,
  "source": "mock"
}
```

**Файлы изменены:**
- `apps/web/src/app/api/load-data/route.ts`

### 4. **Исправления тестов**

**Проблема:**
- Тесты не проходили из-за изменений в API форматах
- Валидация voyage была слишком строгой

**Решение:**
- Обновлены тесты для соответствия новым форматам ответов
- Изменена валидация voyage с 4 символов на 1-4 символа
- Исправлены ожидания в тестах

**Файлы изменены:**
- `apps/web/src/lib/testing/api-test-suite.ts`
- `apps/web/src/app/api/__tests__/ports-search.test.ts`
- `apps/web/src/app/api/__tests__/deadlines.test.ts`
- `apps/web/src/lib/__tests__/deadlines.test.ts`
